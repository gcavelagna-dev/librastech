# **App Name**: LibrasTech

## Core Features:

- Profile Creation: Welcome screen with name input field and 'Save' button for user profile creation.
- Emergency Selection: Display emergency type options (Fire/Rescue, Medical, Public Safety) with representative images for easy understanding.
- Automated SOS Message Generation: Generate a message including the user's name and location to provide the emergency service.
- Navigation: Provide navigation buttons ('Next,' 'Back,' 'Configuration') to move between steps and access settings.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5), reminiscent of safety and reliability, will be used across main interactive elements.
- Background color: Light blue-gray (#ECEFF1), a soft neutral to ensure comfortable readability and accessibility.
- Accent color: A warm orange (#FFAB40) to highlight important interactive elements.
- Headline font: 'Poppins' sans-serif. Body font: 'Poppins' sans-serif. Using the same font provides a uniform experience to the user.
- Use clear, representative icons for emergency categories, adhering to accessibility standards.
- A simple, direct layout to minimize steps in requesting assistance; ensure sufficient touch target sizes.
